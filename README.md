## [Area of Patrol][5mlink]



This simple script will display the Area of Patrol (aka Area of Play, AOP) as well as the State of Emergency (SOE) for three zones: Los Santos (LS), Blaine County (BC) and Paleto Bay (PB).

This script is easily configurable with a user-friendly config file.



## Features
- AOP display
- SOE display for LS, BC and PB
- Possibility to set the AOP to whatever you want using commands
- Easily change the SOE states with commands
- Customize the commands to whatever you'd like
- Restrict commands using ACE permissions
- Auto change the AOP based on player count (toggleable)
- Custom spawn points based on AOP (toggleable)
- Customizable display text and notifications
- Possibility to change the SOE zones (default: LS, BC, PB, but can be changed to whatever you'd like) 
- Fully commented and self-explainatory config file
- Automated version checking (based on this repo)

## Installation
Just drag and drop the folder in your resources. Please make sure that the folder is named "AOP-SOE".

Edit your config.cfg to add the following lines:
```
ensure AOP-SOE
exec @AOP-SOE/permissions.cfg
```

Restart your server.
